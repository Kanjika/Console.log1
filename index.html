What is Visual Studio?
Visual Studio is an integrated development environment from Microsoft. It is used to develop computer programs, as well as websites, web apps, web services and mobile apps.The special part is that it can run offline so you can code wherever and whenever you want!

What is console screen and why is it used? Also how to open the console screen and how to print anything on it?
Console screen is a screen where you can see all the output, it is used to have a better point of view of the code.
To open the console screen follow the steps
$ Click on Go live
$ Right click on the opened screen and click on Inspect
$ Now click on console on the top
